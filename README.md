# DrawReferencer
A small program to display an image of your chosing on top of everything, you can change image and close the software by right clicking on the image

## Usage
Run the exe file if you downloaded that from the release section and choose your image, its that simple


## INSTALL IF RUNNING NON EXE FILE:
* RUN CMD IN THE FOLDER AND RUN: <br>
<code> pip install -r requirements.txt </code>
